# Инструкция по запуску

## Проверка Docker и Docker Compose

Чтобы запустить проект, ничего сложного делать не нужно - всё работает через Docker. Главное, чтобы Docker и Docker Compose были установлены. Проверить, что они стоят и работают, можно обычными командами:

docker --version
docker compose version

## Запуск проекта через Docker Compose

Теперь самое важное - запуск. Он делается одной командой через Docker Compose:

docker compose up --build

Копировать код

После этого Docker сам соберёт образ, поднимет контейнер и запустит backend. Когда всё соберётся, сервис станет доступен по адресу, который прописан в docker-compose.yml. Чаще всего это:

http://localhost:8080
